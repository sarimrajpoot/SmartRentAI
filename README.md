An AI-powered Car Rental & Fleet Management Platform built with FastAPI, React, PostgreSQL, and Artificial Intelligence.

</p>

---

## 🌟 Overview

SmartRentAI is an intelligent vehicle rental platform designed to modernize the traditional rent-a-car industry.

Unlike conventional rental systems, SmartRentAI integrates Artificial Intelligence into every stage of the rental lifecycle, enabling smarter fleet management, enhanced customer experience, and improved road safety.

The platform connects customers with vehicle showrooms across Pakistan while providing advanced AI-powered monitoring, damage detection, GPS tracking, and intelligent recommendations.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/SmartRentAI.git

cd SmartRentAI
```

---

## Backend

```bash
cd backend

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

alembic upgrade head

uvicorn app.main:app --reload
```

Backend runs on

```
http://localhost:8000
```

Swagger

```
http://localhost:8000/docs
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

Frontend runs on

```
http://localhost:5173
```
